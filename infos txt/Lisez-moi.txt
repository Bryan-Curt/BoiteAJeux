Développée par Bryan Curt, Valentin Delpy, Gelel Rezig.

Lancer l'application: ouvrir le dossier du projet, aller dans le dossier dist et lancer le fichier BoiteaJeux.jar.
Il peut aussi être lancé à partir d'une interface de commande type terminal.
Interface connexion : 

Dans un premier temps, cliquer sur le bouton "S'enregistrer" pour vous créer un compte. Cela n'est nécessaire qu'au
premier lancement. Si vous ne souhaitez pas vous inscrire, cliquez sur le bouton "Jouer en tant qu'invité".
Une fois inscrit, connectez vous via le bouton "Connexion". Vous accédez alors à l'interface de choix de jeu.
Cliquez sur le jeu qui vous intéresse.
Le bouton "Score" est aussi disponible, pour obtenir les différentes statistiques de jeu.

Pendu : 

Une fois le jeu lancé, cliquez sur les lettres afin d'essayer de deviner le mot. Vous avez un nombre limité d'essais 
(7 essais maximum). Si vous trouvez le mot, vous avez gagné et aurez alors le choix de rejouer.
Il en va de même pour la défaite.

Bataille navale :

Bataille navale entièrement jouable à la souris. Jeu contre ordinateur intégré.
Placement des bateaux intégré. Possibilité de le faire horizontalement ou verticalement.
Mécaniques de tirs développées. Touché / coulé intégré. Système de victoire / défaite intégré.
Système pour relancer une partie intégré. Possibilité d'abandonner la partie. Un seul niveau de difficulté pour l'IA.

A ajouter en plus : plusieurs niveaux de difficulté.

Pour jouer, vous devez d'abord effectuer le placement de vos bateaux sur la grille joueur. Vous placerez
tour à tour les bateaux de taille 2 à 5. Une fois ce placement effectué, un message s'affiche. Il vous
propose de jouer les tours en cliquant sur la grille de l'ordinateur : vous êtes alors dans la phase de
jeu. Lorsque tous les bateaux sont coulés, vous avez la possibilité de rejouer, ou d'arrêter.

A tout moment, vous pouvez faire le choix d'abandonner la partie, en appuyant sur le bouton prévu à
cet effet en dessous de la grille du joueur.
